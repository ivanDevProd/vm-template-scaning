How to Submit System Image for Analysis

To initiate the analysis process, start by uploading your system image using one of the following methods:

Upload Options:

  • URL Upload: Provide the direct URL to the system image you wish to analyze. Ensure the URL is accessible and contains the correct file format.
  • Local File Upload: Select and upload the file directly from your device. Accepted file formats include .img, .iso, qcow, or .tar.gz.


After Upload:

  • Triggering the Scan Job: Once the image is uploaded, a scan job will automatically begin in a separate process. This job will analyze the image for compliance, verifying that it meets necessary requirements for further deployment.
  • Upload to Artifactory: The image is uploaded to Artifactory for easy access and future use.
  • End-User Email Notification: An email containing details about your image processing status and Jira case will be sent to you, along with any relevant links and information.
  • Jira Case: A Jira case is automatically created to track the image's scan process, helping with progress updates and issue management.

Each of these actions occurs automatically, allowing you to track your image processing progress in real time and receive relevant notifications.
